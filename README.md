⌨️ Easy Typing Game (Myanmar & English)

မြန်မာစာနှင့် အင်္ဂလိပ်စာ ရိုက်နှက်ခြင်း စွမ်းရည်ကို ပျော်ပျော်ပါးပါး လေ့ကျင့်နိုင်မည့် Typing Game တစ်ခု ဖြစ်ပါသည်။ အထူးသဖြင့် မြန်မာစာ ရိုက်ရာတွင် အခက်အခဲရှိသူများအတွက် Game ကစားရင်း လက်ကွက်ကျင့်နိုင်ရန် ရည်ရွယ်ပါသည်။

🌟 Features (အဓိက ပါဝင်သော လုပ်ဆောင်ချက်များ)

    Dual Language Support: မြန်မာစာ နှင့် အင်္ဂလိပ်စာ နှစ်မျိုးလုံးကို စိတ်ကြိုက်ရွေးချယ် ကစားနိုင်ပါသည်။

    Difficulty Levels: Easy, Medium, နှင့် Hard ဆိုပြီး အဆင့် ၃ ဆင့် ပါဝင်ပြီး Language အလိုက် အမြန်နှုန်းများကို ချိန်ညှိထားပါသည်။

    Adaptive UI: ခေတ်မီပြီး မျက်စိပဒေသာ ဖြစ်စေမည့် Glassmorphism UI Design ကို အသုံးပြုထားပါသည်။

    Responsive Design: Laptop နှင့် Desktop များတွင် အဆင်ပြေပြေ ကစားနိုင်ရန် ဖန်တီးထားပါသည်။

    Game Mechanics:

        Score နှင့် Killed count စနစ်။

        Heart (Lives) စနစ် (မှားရိုက်ပါက သို့မဟုတ် စာလုံးများ ကျော်သွားပါက အသက်လျော့မည်)။

        Pause/Resume နှင့် Restart လုပ်ဆောင်ချက်များ။

🛠️ Tech Stack (အသုံးပြုထားသော နည်းပညာများ)

    Framework: React.js (Vite)

    Styling: Tailwind CSS (v4)

    Animation: Framer Motion

    Icons: Lucide-React

    Fonts: Noto Sans Myanmar (Google Fonts)
